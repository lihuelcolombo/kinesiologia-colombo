# kinesiologia-colombo
Proyecto Coder House 
_______

Proyecto Final para el curso de Desarrollo Web de Coder House.

La idea del proyecto fue desarrollar una página que luego pueda ser realmente utilizada para brindar información y a la vez promocionar servicios de mi principal profesión al día de hoy, que es la kinesiología.

El foco estuvo en desarrollar una interfaz simple, intuitiva, y fácilmente navegable, con colores que transmitan seriedad (sin ser rigurosamente formal), seguridad y tranquilidad, por tratarse de una temática de salud.

_______

Versiones de frameworks y utilidades:
Boostrap 5.1.3
Node.js v16.13.0
